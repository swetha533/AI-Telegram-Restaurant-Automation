# 🤖 AI-Powered Telegram Restaurant Automation System

An automated restaurant ordering and customer assistance system built using **n8n**, **Google Gemini AI**, and **Telegram Bot API**.

## 📌 Project Overview
This project automates customer support, menu queries, and order management for restaurants using an AI agent workflow designed in n8n.

## 🛠️ Tech Stack & Tools
- **Workflow Automation:** n8n
- **AI Model:** Google Gemini Chat Model
- **Messaging Integration:** Telegram API
- **Data Format:** JSON Workflow

## ⚡ Key Features
- 24/7 automated interactive responses for restaurant queries.
- Smart AI Agent to handle complex customer interactions naturally.
- Real-time message processing via custom Webhooks.

## 🚀 How to Import & Use
1. Download the `Swetha T Workflow.json` file from this repository.
2. Open your **n8n** dashboard.
3. Click on the 3 dots menu -> **Import from file...**
4. Select the `.json` file and connect your own Telegram Credentials and Gemini API Key.
